# s1_s2 (S1S2ClassifierPlugin)

> Agent-first 插件契约文档。面向自动化执行与改动评估。

## Agent Contract

| Item | Value |
|------|-------|
| Provides | `s1_s2` |
| Depends On | `waveform_width`, `basic_features` |
| Output Kind | `structured_array` |
| Version | `0.3.0` |
| Module | `waveform_analysis.core.plugins.builtin.cpu.s1_s2_classifier` |
| Accelerator | `cpu` |

## Inputs

- `waveform_width`
- `basic_features`

## Config

| Name | Type | Default | Note |
|------|------|---------|------|
| `width_unit` | `str` | `ns` | Width unit used for range checks: 'ns' or 'samples'. |
| `s1_width_range` | `tuple` | `None` | S1 width range (min, max) in width_unit. None disables. |
| `s2_width_range` | `tuple` | `None` | S2 width range (min, max) in width_unit. None disables. |
| `s1_area_range` | `tuple` | `None` | S1 area range (min, max). None disables. |
| `s2_area_range` | `tuple` | `None` | S2 area range (min, max). None disables. |
| `s1_height_range` | `tuple` | `None` | S1 height range (min, max). None disables. |
| `s2_height_range` | `tuple` | `None` | S2 height range (min, max). None disables. |
| `conflict_policy` | `str` | `unknown` | How to resolve when both S1 and S2 criteria match. |
| `strict` | `bool` | `False` | Raise if no S1/S2 criteria are configured. |

## Execution Path

`s1_s2` 依赖链入口：
`waveform_width -> basic_features -> s1_s2`

## Failure Modes

- 依赖数据缺失或字段不匹配，导致 compute 阶段报错
- 配置值类型/范围不合法，触发参数校验异常
- 输出 dtype 变更但版本未升级，可能导致缓存命中异常

## Change Playbook

1. 修改 `options`/`output_dtype`/核心算法后同步提升 `version`
2. 保持 `provides` 稳定；若必须变更，更新依赖插件与文档索引
3. 新增/删除输出字段时，同时更新消费方插件和回归测试
