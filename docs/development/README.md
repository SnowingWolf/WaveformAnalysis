## 🎓 开发工作流

### 开发新功能（1-2 小时）

```
1. 阅读架构文档 - 了解系统设计
   ↓
2. 查看现有实现 - 学习最佳实践
   ↓
3. 编写代码 - 遵循代码规范
   ↓
4. 编写测试 - 确保质量
   ↓
5. 提交代码 - 遵循提交规范
```
